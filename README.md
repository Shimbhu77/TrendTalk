# React Text Transformation App

<!-- ![Preview](preview.png) -->

This is a React app built with React and Bootstrap that allows users to perform various text transformations and analyze text statistics. 
The app supports dark and light modes and provides features such as converting text to lowercase and uppercase, copying text, removing extra spaces,
reversing text, and displaying word and character counts. Users can enter text to analyze and view statistics such as reading time.
The app is deployed on Netlify.

## Features

- Dark and light modes for a personalized viewing experience.
- Text transformation options:
  - Convert text to lowercase and uppercase.
  - Remove extra spaces from the text.
  - Reverse the order of characters in the text.
  - Copy the transformed text to the clipboard.
- Text analysis:
  - Display the word count and character count of the entered text.
  - Estimate the reading time based on the word count.
- Preview options:
  - View a live preview of the entered text.
- Responsive design:
  - The app is designed to work well on different screen sizes and devices.

## Technologies Used

- React: A JavaScript library for building user interfaces.
- Bootstrap: A popular CSS framework for creating responsive and mobile-first web applications.

## Usage

1. Clone the repository:

```shell
git clone [Github](https://github.com/Shimbhu77/Word-Counter.git)
```

2. Install dependencies:

```shell
npm install
```

3. Run the app:

```shell
npm start
```

4. Open your browser and visit [http://localhost:3000](http://localhost:3000) to see the app in action.
